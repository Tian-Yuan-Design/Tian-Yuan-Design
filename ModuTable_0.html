ROLE

In this program, we want to create a movable workplace for library. The manager of Schmidt Hammer Lassen Architects is invited to introduce their design philosophy, production and so on. At last, we dedsigned a set of furniture for SHL. To fit with the main values of SHL, we find some keywords: Scandinavian, Sustainable, Open&White, Functional, Simple.

· Get Inspiration ·

We use moodboard to express the feeling of our ideal products.

· To Create An Inspiring Atomosphere And Efficient Workplace ·

We analyse students' needs in this situation through Maslow Theory. And we hope pay more attention to maintain GOAL FUN SOCIAL EXTENSION, besides of basic function,like storage and own space.

Same Place ---- Different Users

Finally, we design a modular table which allow you to group different shape table togather, so that to create various space. Also, we attach "elastic strings" on board in middle of the table. People can fix their books, pens, toys via the elastic strings.
